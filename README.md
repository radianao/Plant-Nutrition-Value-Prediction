# Plant-Nutrition-Value-Prediction
Implemented Machine Learning algorithms to predict values of plant nutrition from Aria Company' Dataset Sample

## Dataset Description
| Columns        | Description                      |
|-----------| ----------------------------------|
| v1 - v8 | Information about variable to predict plant nutrition |
| sample_type | Information about 2 different labs sample obtained |
| target   | Information to predict plant nutrition |

## Implemented Machine Learning Algorithms
1. Linear Models      : 
   * Linear Regression
   * Support Vector Regression
   
2. Non Linear Models  :
   * KNeighbors Regression
   * Decission Tree Regression
   * Random Forest Regression
   * XGBoost Regression
